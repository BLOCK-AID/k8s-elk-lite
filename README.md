# k8s-elk-lite
Kubernetes Elasticsearch deployment to AWS

The purpose of this project is to provide starter files for deploying a high performance Elasticsearch cluster on Kubernetes running on AWS. 

## K8S-AWS Cluster

### Topology
  + 3 K8S Master Nodes
  + 5 K8S Data Nodes
    ++ 3 ELK Master Nodes
    ++ 2 ELK Data Nodes
